# CapyPIV

## Introduction

CapyPIV is a simple web application for performing Particle Image Velocimetry (PIV) analysis using OpenPIV. This tool provides an easy-to-use interface for analyzing fluid flow patterns through particle tracking.

## Usage

To run the application:

```bash
python app.py
```

Then open your web browser and navigate to the local server address displayed in the terminal.
